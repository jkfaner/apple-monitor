# 苹果库存监控（Apple Monitor）

> 本项目基于远古技术swing和新势力spring IOC容器。（数据均来自apple.com）

[![GitHub stars](https://img.shields.io/github/stars/jkfaner/apple-monitor.svg)](https://github.com/jkfaner/apple-monitor)

### 目前已经支持的功能
+ 自定义地区、自定义机型、自定义监控时间
+ 主题自定义（亮色、暗色、深紫色）

### 计划中支持的功能
+ Bark推送
+ 到货音乐提醒
+ 多地区支持
+ 多模式选择
